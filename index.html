HTML

HTML- HyperText Markup Language

Basic HTML Tags and usage

TAG DISCRIPTION
<!DOCTYPE html> Declaration specifying the HTML version.
<html> The root element of an HTML page..
<head> Contains meta-information about the HTML document.
<body> Contains the content of the HTML document
<title> For adding title on the browser title bar.
<h1> to <h6> Headings, where <h1> is the largest and <h6> is the smallest.
<p> Paragraph: for adding paragraph text
<a> Anchor for creating hyperlinks.
href Attribute specifying the URL
<img> Image: for adding image
src Attribute specifying the image source.
alt Attribute specifying alternative text for the image.
<ul> Unordered list.
<li> List item within an unordered or ordered list.
<ol> Ordered list.
<br> Line break.
<hr> Horizontal rule: for making horizontal line on webpage
<strong> Strong importance (typically bold) for making text bold
<em> Emphasis (typically italic). for making text Italic
<span> Inline container for styling
<div> Block-level container for styling and layout.
<input> Input field for forms.
type Attribute specifying the type of input (text, checkbox, radio, etc.).
value Attribute specifying the initial value..
<textarea> Multiline text input for forms.
<form> Form container
action Attribute specifying where the form data should be sent.
method Attribute specifying the HTTP method for sending form data (e.g., "get" or "post").
<select> Dropdown menu for forms.
<option> Option within a <select>

CSS

CSS stands for Cascading Style Sheets. It is a style sheet language used for describing the look and formatting of a document written in a markup language like HTML.

Basic CSS Tags and usage

TAG DISCRIPTION
Body: body{ /* Styles for the entire document body */ font-family: Arial, sans-serif; background-color: #f0f0f0; }
Headings (h1, h2, h3, etc.): h1 { /* Styles for the first-level heading */ color: #333; }
h2 { /* Styles for the second-level heading */ color: #555; }
/* Add styles for other heading levels as needed */
Paragraphs (p): p { /* Styles for paragraphs */ line-height: 1.5; margin-bottom: 10px; }
Links (a): a { /* Styles for links */ color: #0066cc; text-decoration: none; }
a:hover { /* Styles for links on hover */ text-decoration: underline; }
Lists (ul, ol, li): ul, ol { /* Styles for unordered and ordered lists */ margin-left: 20px; }
li { /* Styles for list items */ list-style-type: disc; /* or 'circle', 'square', etc. */ }
Images (img): img { /* Styles for images */ max-width: 100%; height: auto; }
Divs and Spans: .container { /* Styles for a class named 'container' */ width: 100%; padding: 20px; margin: 0 auto; }
#uniqueElement { /* Styles for an element with the ID 'uniqueElement' */ color: #ff0000; }
Color Property: "The color property sets the text color"
p { color: #333; /* Hex color code */ }
Background Color: The background-color property sets the background color of an element.
body { background-color: #f0f0f0; /* Hex color code */ }
Border Color: The border-color property sets the color of the borders.
div { border: 1px solid #999; /* Hex color code */ }
Opacity: The opacity property controls the transparency of an element.
.transparent-box { background-color: #aabbcc; opacity: 0.8; /* Ranges from 0 (fully transparent) to 1 (fully opaque) */ }
RGBA Color: The rgba function allows you to specify a color using red, green, blue, and alpha (transparency) values.
h2 { color: rgba(255, 0, 0, 0.7); /* Red color with 70% transparency */ }
HSL Color: The hsl function allows you to specify a color using hue, saturation, and lightness values.
a { color: hsl(120, 100%, 50%); /* Green color */ }
Text Shadow: The text-shadow property adds a shadow to text.
h1 { text-shadow: 2px 2px 4px #666; /* Horizontal offset, vertical offset, blur radius, color */ }
Box Shadow: The box-shadow property adds a shadow to an element.
div { box-shadow: 3px 3px 5px #888; /* Horizontal offset, vertical offset, blur radius, color */ }
Font Family: The font-family property defines the font to be used for text.
body { font-family: "Arial", sans-serif; }
Font Size: The font-size property sets the size of the text.
p { font-size: 16px; }
Font Weight: The font-weight property sets the thickness of the characters.
strong { font-weight: bold; }
Font Style: The font-style property sets whether the text is italic or normal.
em { font-style: italic; }
Text Decoration: The text-decoration property controls underlining, overlining, and striking through text.
a { text-decoration: none; /* No underline */ }
.underline { text-decoration: underline; }
Text Transform: The text-transform property controls the capitalization of text.
h1 { text-transform: uppercase; }
Line Height: The line-height property sets the height of a line of text.
p { line-height: 1.5; }
Letter Spacing: The letter-spacing property controls the space between characters.
h2 { letter-spacing: 2px; }
Text Align: The text-align property sets the horizontal alignment of text.
div { text-align: center; }
Display Property: display: block;: Makes an element a block-level element, taking up the full width available.
display: inline;: Makes an element an inline-level element, allowing other elements to sit beside it.
display: inline-block;: Combines features of both block and inline elements.
Position Property: position: static;: Default value. Elements are positioned according to the normal flow of the document.
position: relative;: Positioned relative to its normal position.
position: absolute;: Positioned absolute to its first positioned (not static) ancestor element.
position: fixed;: Positioned relative to the browser window.
position: sticky;: Acts like relative within its container until it crosses a specified point, then it is treated as fixed
Flexbox: Flexbox is a one-dimensional layout method for laying out items in rows or columns. It's great for distributing space along a single axis.
display: flex;
flex-direction: row | column;
justify-content: flex-start | flex-end | center | space-between | space-around;
align-items: stretch | flex-start | flex-end | center | baseline;
Grid: Grid is a two-dimensional layout method, providing both row and column-based layout capabilities.
display: grid;
grid-template-rows: auto | 1fr;
grid-template-columns: auto | 1fr;
grid-gap: 1em;
Float Property: float: left; or float: right;: Elements are shifted to the left or right, allowing text and inline elements to wrap around them. (Note: Flexbox and Grid are generally preferred over float for layout purposes in modern CSS.)
Box Model: margin, padding, border: Properties for controlling the spacing and borders around elements.
margin: 10px;
padding: 10px;
border: 1px solid #000;
Responsive Design: Media queries are often used for creating responsive layouts that adapt to different screen sizes.
@media (max-width: 600px) { /* CSS rules for screens smaller than 600px */}

Usage of Tags with Examples

HTML & CSS

"Giving Title to Browser Title Bar"

To give a title to the browser title bar in an HTML document, you need to use the <title> element within the <head> section of your HTML document. The text you put inside the <title> element will be displayed on the title bar of the browser.

    <!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Your Page Title Here</title>
</head>
<body>
    <!-- Your page content goes here -->
</body>
</html>

"To add an image to a website"

To add an image to a website, you can use the <img>(image) tag in HTML. The <img> tag is a self-closing tag, meaning it doesn't have a closing tag. It includes the src attribute, which specifies the URL (file path) of the image, and optionally, you can include attributes such as alt for alternative text, width and height to define the dimensions, and more.

    <!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Image Example</title>
</head>
<body>

    <!-- Adding an image to the website -->
    <img src="path/to/your/image.jpg" alt="Description of the image" width="300" height="200" style="border: 1px solid #ddd;">

</body>
</html>

"To add an video to a website"

You can use the HTML5 <video> element to embed a video in your HTML document and provide controls using the controls attribute. Here's a simple example:

    <!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Video with Controls</title>
</head>
<body>

<h1>Video Example</h1>

<video width="640" height="360" controls>
    <source src="your-video-file.mp4" type="video/mp4">
    Your browser does not support the video tag.
</video>

</body>
</html>
